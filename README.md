# JARVIS-AGATA 🤖

Asistente de IA personal con interfaz holográfica para Windows.
Powered by Google Gemini Live API.

## Características
- Conversación por voz en tiempo real
- Control del sistema (apps, archivos, pantalla)
- Búsqueda web y noticias
- Control de Spotify y YouTube
- Monitor del sistema
- Gestión de archivos y PDFs
- Dos personalidades: JARVIS y AGATA

## Requisitos
- Python 3.13+
- API Key de Google Gemini (gratuita en aistudio.google.com)

## Instalación
```bash
pip install -r requirements.txt
python main.py
```

## Configuración
Al primer inicio se pedirá la Gemini API Key.
